# Block Blast (Python / tkinter)

A lightweight Block Blast-style puzzle game implemented in pure Python using `tkinter`.

## Run

```bash
python3 block_blast.py
```

## Rules

- Select one of the 3 available pieces.
- Click a board position to place it.
- Fill an entire row or column to clear it and score bonus points.
- When all 3 pieces are used, you get a fresh set.
- The game ends when no available piece can be placed.
